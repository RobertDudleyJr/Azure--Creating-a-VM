<h1>Microsoft Azure - Creating a Virtual Machine (VM)</h1>
This tutorial outlines the prerequisites and installation steps for configuring a virtual machine using Microsoft Azure. Virtual Machines are great for testing!


<h2>Video Demonstration</h2>

<p>N/A</p>

<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)



<h2>Operating Systems Used </h2>

- Windows 10</b> (21H2)

<h2>List of Prerequisites</h2>

- Microsoft Azure Subscription Needed.
- Resource Group needs to be created first.  

<h2>Confirguration Steps</h2>

<p>
<img src=https://static.wixstatic.com/media/695ae6_6395fe8a45d14b578c88c43509893730~mv2.png/v1/fill/w_600,h_340,al_c,q_85,usm_0.66_1.00_0.01,enc_auto/VM-1.png" height="80%" width="80%" alt="Step1"/>
</p>
<p>
Log in to your Microsoft Azure Account and type resource groups in the search box. Research groups should pop up under Services. Click on it.    
</p>
<br />

<p>
<img src="https://static.wixstatic.com/media/695ae6_38f03e82125247a8a8a97c83c59751f4~mv2.png/v1/fill/w_600,h_244,al_c,q_85,usm_0.66_1.00_0.01,enc_auto/VM-2.png" height="80%" width="80%" alt="step2"/>
</p>
<p>
Click the Create Tab to create a new Resource Group. A Resource group must be created before a Virtual Machine can be created.
</p>
<br />

<p>
<img src="https://static.wixstatic.com/media/695ae6_49c343fae18947e584e59ebbdb4b192c~mv2.png/v1/fill/w_600,h_337,al_c,q_85,usm_0.66_1.00_0.01,enc_auto/VM-3.png" height="80%" width="80%" alt="Step3"/>
</p>
<p>
As a prerequisite, you should have created a subscription account. Click the drop-down and select the subscription account you would like to charge. Next, create a Resource group name. Last, select the Region closets to your location. 
</p>
<br />
<p>
<img   src="https://static.wixstatic.com/media/695ae6_d2f0042de7d743029c3b9e9d917c0e0f~mv2.png/v1/fill/w_600,h_355,al_c,q_85,usm_0.66_1.00_0.01,enc_auto/VM-4.png" height="80%" width="80%" alt="Step4"/>
</p>
<p>
Review everything and go ahead and click create. 
</p>
<br />
</p>
<br />
<p>
<img src="https://static.wixstatic.com/media/695ae6_41fa4acc03aa4b00a88512e548d204df~mv2.png/v1/fill/w_600,h_341,al_c,q_85,usm_0.66_1.00_0.01,enc_auto/VM-5.png" height="80%" width="80%" alt="Step5"/>
</p>
<p>
Next, we need to create a Virtual Machine (VM). Type Virtual Machines in the search bar and select Virtual Machines. 
</p>
<br />

<p>
<img src="https://static.wixstatic.com/media/695ae6_096aaa3d45ee40aeaa1a7f4a35e402f7~mv2.png/v1/fill/w_600,h_351,al_c,q_85,usm_0.66_1.00_0.01,enc_auto/VM-6.png" height="80%" width="80%" alt="step6"/>
</p>
<p>
Click Virtual Machine again. 
</p>
<br />

<p>
<img src="https://static.wixstatic.com/media/695ae6_4570a7f6ab14479289b4e321d47425bd~mv2.png/v1/fill/w_600,h_339,al_c,q_85,usm_0.66_1.00_0.01,enc_auto/VM-7.png" height="80%" width="80%" alt="Step7"/>
</p>
<p>
Now you will be prompted to select your subscription, select the Resource group you created, and create a Virtual Machine name.
</p>
<br />
<p>
<img   src="https://static.wixstatic.com/media/695ae6_7057459bdf854012a51787d0b87d973a~mv2.png/v1/fill/w_600,h_373,al_c,q_85,usm_0.66_1.00_0.01,enc_auto/VM-8.png" height="80%" width="80%" alt="Step8"/>
</p>
<p>
Select your virtual image (Operating System) and your memory size 
</p>
<br />
</p>
<br />
<p>
<img src="https://static.wixstatic.com/media/695ae6_ec7a5075091d45c8822eaeb74728d616~mv2.png/v1/fill/w_600,h_363,al_c,q_85,usm_0.66_1.00_0.01,enc_auto/VM-9.png" height="80%" width="80%" alt="Step9"/>
</p>
<p>
Create a Username and Password for your virtual machine.
</p>
<br />

<p>
<img src="https://static.wixstatic.com/media/695ae6_29665f134a994ad1be0c733010f3773a~mv2.png/v1/fill/w_600,h_337,al_c,q_85,usm_0.66_1.00_0.01,enc_auto/VM-10.png" height="80%" width="80%" alt="step10"/>
</p>
<p>
Make sure you check the check box for Licensing.
</p>
<br />

<p>
<img src="https://static.wixstatic.com/media/695ae6_4c55d5c0304147068e4587126c18fa52~mv2.png/v1/fill/w_600,h_418,al_c,q_85,usm_0.66_1.00_0.01,enc_auto/VM-11.png" height="80%" width="80%" alt="Step11"/>
</p>
<p>
If your virtual machine has passed validation, you can now go ahead and click Create.
</p>
<br />
<p>
<img   src="https://static.wixstatic.com/media/695ae6_e6b78001e1f943e9b4385c45d7a4af9b~mv2.png/v1/fill/w_600,h_319,al_c,q_85,usm_0.66_1.00_0.01,enc_auto/VM-12.png" height="80%" width="80%" alt="Step12"/>
</p>
<p>
Your deployment is complete. You have now created a virtual Machine! 
</p>
<br />
</p>
